Cómo usar la herramienta
1. Seleccionar archivos
hacer clic en el botón “Seleccionar archivos” y eligir uno o varios archivos .txt que se desee comprimir.

2. Elegir un algoritmo
En la sección de algoritmo, seleccionar el algoritmo de compresión que se quiere utilizar.
-Huffman
-LZ77
-LZ78

4. Comprimir
Presionar el botón “Comprimir”.
Se abrirá una ventana para elegir la ubicación donde se guardará el archivo comprimido.

4. Asignar nombre
Guardar el archivo con un nombre siguiendo esta recomendación:
{AlgoritmoUtilizado}_{NombreOriginal}.myzip


Ejemplo:
Huffman_PruebaHuffman.myzip

5. Descomprimir
Para descomprimir un archivo, haz clic en “Descomprimir”, selecciona el archivo .myzip y la herramienta generará los archivos originales dentro de una carpeta.
